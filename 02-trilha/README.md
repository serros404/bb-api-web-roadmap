# 02 · Trilha — as fases F0 → F7

[🏠 Início](../README.md)

A trilha do roadmap — mas **não é uma fila**. A **espinha** (F0→F3) constrói habilidade em ordem; as **frentes contínuas** (F4 CTF/labs, F5 CVE) rodam **em paralelo, desde cedo**; F6/F7 são conversão e sustentação **contínuas**. Veja [como as camadas rodam no tempo](../README.md#-as-fases-e-como-elas-rodam-importante-não-é-uma-fila). Prazos **indicativos** (~15–20h/sem) — capacidade demonstrada > tempo decorrido.
